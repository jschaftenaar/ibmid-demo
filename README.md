# IBMid demo

## Requirements

* [PHP](http://php.net) >= 5.6

## Installation

1. Download PHP dependencies via Composer:

  ```shell
  composer install
  ```

## Features

This demo app was built for testing the IBMid (Single Sign On authentication service for Bluemix)
registration flow.
