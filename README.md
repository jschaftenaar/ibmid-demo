# IBMid demo

## Requirements

* [PHP](http://php.net) >= 5.6

## Installation

1. Download PHP dependencies via Composer:

  ```shell
  composer install
  ```
2. Set required environment variables:

  * `IBMID_ENDPOINT_BASE_URL`
  * `IBMID_CLIENT_ID`
  * `IBMID_CLIENT_SECRET`

3. Set `APP_URL` in `\App\Oauth2` class
